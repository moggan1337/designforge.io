# DesignForge.io

> AI-powered design system generator. Describe your product, get production-ready code.

![DesignForge](https://img.shields.io/badge/DesignForge-AI%20Design-brightgreen)
![Next.js](https://img.shields.io/badge/Next.js-14-black)
![Tailwind](https://img.shields.io/badge/Tailwind-CSS-blue)

## Features

- **AI-Powered Generation** - Describe your product and get a complete design system
- **Tailwind Config** - Custom colors, fonts, and spacing ready to copy
- **shadcn/ui Components** - Production-ready React components
- **CSS Variables** - Easy theming with custom properties
- **Dark Mode Ready** - Automatic dark mode implementation
- **Multi-Framework Export** - React, Vue, HTML support

## How It Works

1. **Describe** - Tell us about your product, audience, and vibe
2. **Generate** - AI creates your complete design system
3. **Export** - Download Tailwind config, components, or CSS variables

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS v4
- **Icons:** Phosphor Icons
- **Fonts:** Geist

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
